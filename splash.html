How to create boot spalshes

This is a tutorial on how to create custom boot splashes and install them using Anemone. Make sure you can access CFW before attempting. These will not work with homebrew only. 
Step 1

Go and download this zip file and extract it.

Step 2

Open `Both Screens.png` with an image editing software like GIMP.

Step 3

Make your desired changes and save as a `png` file whereever you want.

Step 4

Now go to the Image to BIN converter and upload and download your BIN file.

You can now either place this BIN file on your SD card in `SD://Luma/splash.bin` or upload the BIN directly to a href="https://themeplaza.eu/">Theme Plaza.
